# PROJETS POUR SITES DE DÉTAIL

## Instructions
Pour créer d'autres pages de détail (comme pour un mapping spécifique), copiez simplement `detail-script.html` et renommez-le (ex: `detail-mapping-sud.html`).

Ensuite, modifiez :
1. Le Titre (`<h1>`)
2. La description
3. Les images (section `.product-gallery`)
4. Le prix
5. Les tags techniques (Lua, 3D, etc.)

## Images
N'oubliez pas d'ajouter vos vraies images dans le dossier et de changer les liens `picsum.photos`.
